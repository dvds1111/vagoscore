04 · Validación histórica

¿Qué tan bueno es el modelo?

El backtesting corre el modelo sobre partidos ya jugados y mide su acierto real. La honestidad del producto vive aquí.

Pega tus predicciones históricas en formato JSON (una lista). Cada partido necesita las probabilidades del modelo, el resultado real y, opcionalmente, las cuotas para calcular el ROI.

VagoScore es una herramienta de análisis estadístico, no un consejo de apuesta ni financiero. Ningún modelo predice el fútbol con certeza. El criterio de Kelly optimiza el crecimiento a largo plazo bajo supuestos que la realidad no siempre cumple.
